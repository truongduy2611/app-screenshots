# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] — 2026-03-18

### Added
- Multi-instance screenshot editing with side-by-side view
- Dynamic overlays — text, image, and icon with rich styling and snap-to-grid
- AI-powered translation for all design elements
- Per-locale layout and style customization system
- Device frame rendering (iPhone, iPad, Mac, Watch, Android, etc.)
- App Store Connect direct screenshot upload
- iCloud sync for designs across devices
- Export as PNG, copy to clipboard, and `.appshots` file sharing
- Full keyboard shortcut support
- Dark and light themes
